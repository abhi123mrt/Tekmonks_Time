# Tekmonks_Time
Start the Spring Boot application
Run the main class. The backend starts on:

http://localhost:8080


Place the HTML file correctly
Put your index.html inside:

src/main/resources/static/


Open the HTML page
Go to:

http://localhost:8080/index.html


Click the button
It calls the API:

http://localhost:8080/getTimeStories


Stories appear on the page
The backend returns the 6 predefined JSON stories, and the HTML displays them.
